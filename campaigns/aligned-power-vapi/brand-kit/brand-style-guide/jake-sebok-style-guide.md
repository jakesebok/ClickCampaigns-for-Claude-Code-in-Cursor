# Jake Sebok Brand Style Guide

*Slate & Spark — Used for VAPI Assessment campaign and all Jake Sebok assets.*

---

## 1. VISUAL IDENTITY

### Colors (Slate & Spark palette)

| Role | Hex | Usage |
|------|-----|-------|
| **Background** | #F5F7FA | Page backgrounds |
| **Dark** | #0E1624 | Primary text, nav |
| **Dark 2** | #192236 | Secondary dark elements |
| **Orange (Primary)** | #FF6B1A | CTAs, accents, highlights |
| **Orange 2** | #FF9F6B | Gradients, lighter accents |
| **Border** | #DDE3ED | Borders, dividers |
| **Mid** | #3A4A5C | Body text, secondary copy |
| **Muted** | #7A8FA8 | Labels, captions |
| **Off** | #EEF1F7 | Toggle backgrounds, subtle surfaces |

### Typography

- **Headings:** Outfit, weight 800 (Extra Bold)
- **Body:** Cormorant Garamond, weight 600 (SemiBold)
- **Eyebrow / Labels:** Cormorant Garamond, 10px, uppercase, letter-spacing 0.22em
- **Buttons:** Outfit, weight 600

**Google Fonts import:**
```
Cormorant Garamond: 600, 700 (italic)
Outfit: 400, 500, 600, 700, 800
```

### Logo

- **Primary:** Horizontal logo (flame + JAKESEBOK) — `logo-jake-sebok-horizontal.png`
- **Colors in logo:** Deep navy (#2D3953), vibrant orange (#EA6C2D)

### Spacing & Layout

- **Base unit:** 4px
- **Container max-width:** 1080px
- **Section padding:** 80–100px vertical

---

## 2. UI COMPONENTS

### Buttons (pill-shaped)

All buttons use **border-radius: 100px** for a full pill shape.

| Style | Specs |
|-------|-------|
| **Primary (fill)** | Orange background, white text, 11px font, 600 weight, letter-spacing 0.06em, padding 13px 26px |
| **Outline (light)** | Transparent bg, dark text, 1.5px border, same padding |
| **Outline (dark)** | Transparent bg, light text, 1.5px border for dark backgrounds |

### Toggle groups (theme, font)

- **Container:** Pill-shaped (border-radius: 100px), light background (#EEF1F7), 3px padding, 1px border
- **Buttons:** Pill-shaped (border-radius: 100px), 10px font, 600 weight
- **Active state:** Orange fill, white text

### Cards

- **Border radius:** 20px (cards), 16px (stat boxes)
- **Padding:** 32px (cards), 28px 24px (stat boxes)
- **Border:** 1.5px solid, hover: orange accent

### Eyebrow / accent rule

- **Rule:** 2px height, orange, 2px border-radius
- **Label:** 10px, uppercase, 0.22em letter-spacing

---

## 3. USAGE NOTES

- **Pill shape (100px radius)** is the signature for all buttons and toggle groups
- Use orange for CTAs, eyebrow labels, and accent highlights
- Dark theme: backgrounds #0c0a09 / #0E1624, light text
- Contrast theme: light base with dark framework sections
- Headlines: Outfit 800, tight letter-spacing (-0.03em to -0.04em)
