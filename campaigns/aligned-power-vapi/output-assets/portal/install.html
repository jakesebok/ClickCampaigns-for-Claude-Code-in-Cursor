Add VAPI to your home screen

  1. Tap the Share icon at the bottom of Safari.
  2. Scroll and tap Add to Home Screen.
  3. Tap Add in the top right corner.

On iPhone, the browser install prompt is manual. After install, VAPI behaves like a native app — push notifications, offline shell, one-tap open.

  1. Tap the menu button in Chrome.
  2. Choose Add to Home screen or Install app.
  3. Confirm Install.

Click the install icon in your browser's address bar (look for a ⬇ in Chrome / Edge).
