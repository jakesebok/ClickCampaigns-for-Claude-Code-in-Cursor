# Webinar & Event Registration — Tool Suggestions

For **Aligned Freedom Workshop** (monthly, free) and **Strategic Alignment Intensives** (quarterly, paid), you'll need a way to:

1. Collect registrations
2. Send automated reminders (email; add other channels if your stack supports them)
3. Host the event (Zoom, etc.) or link to a scheduler

## Recommended Tools

### For the Freedom Workshop (free, recurring)

| Tool | Best For | Notes |
|------|----------|-------|
| **Calendly** | Simple signup + reminders | Free tier works. Create a "group event" for the workshop. Sends reminders automatically. |
| **Zoom Webinars** | Built-in registration | Free with Zoom account. Registration form + email reminders included. Good for 100+ attendees. |
| **ConvertKit** | Email + webinar in one | If you already use ConvertKit, use a landing page + form + broadcast for reminders. |
| **Kajabi / Teachable** | All-in-one | If you have a course platform, add a "live workshop" product with registration. |

### For Strategic Alignment Intensives (paid, quarterly)

| Tool | Best For | Notes |
|------|----------|-------|
| **Calendly + Stripe** | Simple paid events | Calendly can collect payment via Stripe for paid events. |
| **Acuity Scheduling** | Scheduling + payments | Handles signup, payment, reminders, and calendar sync. |
| **HoneyBook** | Full client workflow | Good if you want proposals, contracts, and payment in one place. |
| **Eventbrite** | Event discovery + ticketing | Good for paid workshops. Handles ticketing, reminders, and optional public listing. |

### Free / Low-Cost Combo

- **Calendly** (free) for signup
- **Zoom** (free) for the event
- **Gmail / ConvertKit** for manual or automated reminders

## Implementation Notes

- The Freedom Workshop and Strategic Intensives landing pages currently link to `/contact` as a fallback until you set up registration.
- Once you choose a tool, update the CTA on each landing page to point to your registration URL (e.g. Calendly link, Zoom registration page, or Eventbrite event).
