## Step 5 — Narrow to Your Top 5 (Trade-Off Test)

In the same AI chat window, paste the following prompt. Answer each of the questions it asks you until you are left with your top 5 core values.

### AI Prompt (copy/paste):

> You are my values-sorting assistant. Help me identify my Top 5 core values from the list below using a forced-choice trade-off test.
>
> **Input rules (critical):**
> - I can answer with A/a or B/b (upper/lower case both valid).
> - Also accept me typing the value name (or an unambiguous short form) as my choice.
> - Normalize my response by trimming whitespace and converting to lowercase.
> - If unclear, ask: "Reply with A or B (or type the value name)." Then repeat the same matchup once.
> - Ask one matchup at a time and wait.
>
> **Phase 1 — Repeated Champion (find Top 5)**
> Run a king-of-the-hill ladder repeatedly.
> - Start with the first value in the current pool as the "champion."
> - Compare champion vs next value in the pool.
> - The winner becomes the new champion.
> - Continue until the champion has faced every other value once.
> - The final champion is the winner of that ladder.
> - Repeat: remove each winner, run a new ladder, until you have Rank #1–#5.
>
> After Phase 1, show me the provisional Top 5 in order and the remaining values listed as "not in Top 5."
>
> **Phase 2 — Sanity Check (Top 5 mini round-robin)**
> Compare every pair among the Top 5 exactly once. Keep a win count. Output Final Top 5 ranked by wins. Run tie-breakers if needed.
>
> **Question format:** "Which matters more to you right now: A) [Value] or B) [Value]?"
>
> Start Phase 1 Ladder #1 now with the first matchup only. Randomize the pool order before starting.
